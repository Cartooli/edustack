# Create Templates — Starter Projects for Students

Pick a template to start a new project. Each one has the HTML, CSS, and JavaScript
structure already set up — you fill in the content and customize the design.

---

## Available Templates

### 1. Personal Portfolio (`templates/starter.html`)
A clean, one-page personal site. Good for practicing HTML and CSS.
**Use after:** Skill-04

### 2. Interactive Quiz (`templates/quiz-template.html`)
A quiz app with questions, scoring, and results. Ready to customize.
**Use after:** Skill-08

### 3. Dashboard (`templates/dashboard-template.html`)
A data dashboard with stat cards and progress bars. Add your own data.
**Use after:** Skill-06

### 4. Blank Canvas (`templates/blank.html`)
Just the basics — a clean HTML file with a style section and script section.
**Use anytime.**

---

## How to Use a Template

1. Find the template file in the `templates/` folder
2. **Copy** it to your working folder (don't edit the original!)
3. Rename it to something meaningful (`my-quiz.html`, `portfolio.html`)
4. Open it and start customizing

**Remember:** Templates are starting points, not finished products.
Make it your own!
