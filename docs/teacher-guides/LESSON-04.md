# Teacher Guide: Lesson 04 — My First Site

> **Prep time: 15 minutes.** This is a project lesson — read the full guide.

---

## What This Lesson Teaches

Students combine everything from Lessons 01–03 to build a complete multi-section
personal website from scratch. They plan on paper before coding, structure with
HTML, style with CSS, and add animations.

## What Students Will Build

A complete personal site with:
- A styled header with their name/nickname
- At least 3 content sections (About, Interests, Learning, etc.)
- CSS styling with custom colors, fonts, and spacing
- At least one animation or transition effect
- A footer

**Expected result:** A polished site they're proud to show others.

## Before Class

- [ ] Computers are on and logged in
- [ ] Blank paper and pencils available for planning sketches
- [ ] You've reviewed the SKILL-04 file to understand the steps
- [ ] Consider: Will you do a gallery walk at the end? Plan the timing.
- [ ] You've decided on the file naming convention (e.g., `my-site.html`)

## Timing (45-minute class)

| Time | Activity |
|---|---|
| 0:00–0:03 | Introduction: "Today you build YOUR website from scratch." |
| 0:03–0:10 | Paper planning: Sketch sections, pick colors, write content ideas |
| 0:10–0:35 | Build time: Students follow SKILL-04 and customize heavily |
| 0:35–0:43 | Gallery walk: Students walk around and view each other's sites |
| 0:43–0:48 | Sharing: 2–3 volunteers show their sites on the projector |
| 0:48–0:50 | Celebrate: "You just built a website. A real one." |

## The Demo That Hooks Them

Show a completed example site (open `examples/css-fun/index.html` or build your own
simple one). Point out: "This entire page is ONE file. No apps, no accounts, no
internet needed. You wrote every line. By the end of class, you'll have one too."

## Common Questions

**"Can I use a different layout than the example?"**
"Absolutely! The example is a starting point. Change the sections, colors, fonts —
make it yours. That's the whole point."

**"How many sections do I need?"**
"At least 3 content sections plus a header and footer. But if you want more, go for it!"

**"Can I add pictures?"**
"We haven't covered images yet, so stick with text, colors, and shapes for now.
You can always add images to this site later."

**"My page doesn't look like the example."**
"Good! It shouldn't. The example is a template. Your page should reflect YOUR style
and YOUR content."

## Common Mistakes

| Mistake | What You'll See | Fix |
|---|---|---|
| Spending too long planning | Not enough build time | Set a 7-minute planning timer |
| Copying the template exactly | Generic-looking site | Encourage customization: "Change at least 3 colors" |
| Forgetting to save | Lost work | Remind: save after every change (Ctrl+S) |
| Too much content, unfinished | Incomplete sections | "Finish 3 great sections rather than 6 empty ones" |

## Safety Notes for This Lesson

- **First names or nicknames only** — no last names, schools, addresses, or photos
- Review paper plans before students start coding to catch inappropriate content early
- Walk the room during build time — check for appropriate text and images
- Gallery walk: remind students to be kind and encouraging about each other's work
- If a student's content includes personal details, quietly redirect: "Let's keep
  it to first name only — your code skills are the star here, not personal info."

---

*This is a milestone lesson. Students leave with a complete website they built
themselves. Celebrate it. Display sites if you can. This is the moment many students
decide they like coding.*
