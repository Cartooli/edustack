# Review Your Work — Student Checklist

Before you share your page or move to the next skill, check these things!

---

## Does It Work?

- [ ] My file is saved (Ctrl+S or Cmd+S)
- [ ] My file ends in `.html`
- [ ] My page opens in the browser without errors
- [ ] All the text I want to show is visible
- [ ] Links, lists, and tables work correctly

## Is It Organized?

- [ ] My code is indented (nested tags are pushed in with spaces or Tab)
- [ ] I have comments explaining the important parts (`<!-- like this -->`)
- [ ] My tags are properly closed (every `<tag>` has a `</tag>`)
- [ ] My CSS is inside a `<style>` tag in the `<head>` section

## Is It Safe?

- [ ] I did NOT include my full name, address, or phone number
- [ ] I did NOT include any email addresses
- [ ] I did NOT include my school name or teacher name
- [ ] I did NOT include links to external websites
- [ ] All my content is appropriate for school

## Does It Look Good?

- [ ] The page is readable (text isn't too small or too big)
- [ ] Colors have enough contrast (dark text on light background or vice versa)
- [ ] There is spacing between sections (not everything crammed together)
- [ ] It looks okay on a wide screen AND a narrow screen

## Am I Learning?

- [ ] I can explain what each HTML tag on my page does
- [ ] I can explain what my CSS properties do
- [ ] If I delete a line and something breaks, I can figure out why
- [ ] I tried at least one thing that wasn't in the instructions
